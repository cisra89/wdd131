Website Planning Document

Site Name

CircuitHub Electronics

I chose this name because it represents a central hub for electronics products such as components, devices, and accessories. The word “Circuit” connects directly to electronics and technology.

Optional domain: pantech-electronics.com

Site Purpose

The website will present an electronics store where users can browse products, view specifications, compare items, and submit a contact or quote request form. The site will help customers discover electronics products and accessories in an organized and responsive layout.

Scenarios

  • Which laptop or component fits my technical needs and budget?
  • How can I contact the store to request availability or pricing?

Color Schema

Primary Color — Tech Blue (#0b3c5d)

Used for headers, navigation, and major accents.

Accent Color — Energy Yellow (#f4b400)

Used for highlights, buttons, and call-to-action areas.

Light Background — Soft Gray (#f5f7fa)

Used for page background.

Typography

Montserrat — Used for headings and logo.

Open Sans — Used for body text and paragraphs.
